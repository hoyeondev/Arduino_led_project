## 📌 프로젝트명
GUI 프로그램으로 LED를 조작하자🎵

<br>

## 💖 목표
- GUI 프로그램을 생성한다.
- GUI로 아두이노를 제어한다.
- 옵션에 맞는 색상으로 LED를 변경한다.
